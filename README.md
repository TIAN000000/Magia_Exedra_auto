运行原理是图像识别的点击。
点击部分：PyAutoGUI
窗体部分：Pyside6

需要使用1280*720分辨率，窗口化运行游戏，画质任意，我测试下来不影响。游戏界面必须在最前面，并且你不能同时干别的，挂机软件不需要在前台也没有遮挡问题。
不能做成和明日方舟的那个挂机一样控制手机，因为这游戏你打开ADB就不让运行了，别和我说什么ROOT之后隐藏ADB啥的，这样的软件做出来也没多少人能用明白

截图目前只有英文版的，我用国际服制作的，日文看不懂不想玩日服。不过国际服和日服运行界面没有区别，做一份日服截图即可直接运行，后面会做。

link raid挂机现在可以从后台把游戏拉到主界面，不需要手动点游戏。
晶花挂机部分还没加入这个功能，需要点启动之后，鼠标点一下游戏窗口内部的界面，否则第一下点击是不触发的。这一部分的功能还比较简单，后面会加的更方便一点。
