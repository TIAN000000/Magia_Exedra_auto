运行原理是图像识别的点击。
点击部分：PyAutoGUI
窗体部分：Pyside6

需要使用1280*720分辨率，窗口化运行游戏，画质任意，我测试下来不影响。游戏界面必须在最前面，并且你不能同时干别的，挂机软件不需要在前台，挂机软件被其他窗口挡住了也问题不大。

注意：如果使用了其他分辨率的截图，那么第一次点击的位置会失效，需要手动点击右下角

这个游戏不能做成手机挂机的软件，即为类似明日方舟挂机。使用ADB截图并点击。因为这游戏你打开ADB就不让运行了，别和我说什么ROOT之后隐藏软件检测ADB啥的，软件门槛过高会导致绝大多数用户无法使用，另外这游戏反作弊及其恶心，会限制脚本使用键盘进行控制。

我自己目前完成了1280*720的截图，国际版和日文均有。切换语言请把文件夹重命名成aim，重命名前记得把原来的改名字。不要把两个文件夹的内容叠加到一起，否则会极大影响效率或者出bug

挂机现在可以从后台把游戏拉到主界面，不需要手动点一下游戏。

link raid挂机需要在游戏主界面启动
晶花挂机需要在选中队伍界面启动，即为点击play后就开始游戏
